# Fake News Detection using BERT and NLP

## Overview

This project is an AI-powered Fake News Detection system that uses Machine Learning, Natural Language Processing (NLP), and BERT-based text classification techniques to identify whether a news article is fake or genuine.

## Features

* Fake vs Real news classification
* NLP preprocessing pipeline
* Machine Learning and BERT models
* Real-time prediction system
* Interactive deployment support
* GitHub-integrated project structure
